Cookies servem para armazenar dados no navegador.

Exemplo:

<?php
$valor = $_COOKIE['usuario'];
setcookie('usuario', 'Fulano');
echo "$valor";
?>



Include inclui um arquivo passado por parametro

Exemplo:

<?php
include_once("esc.html");
?>



Function pode ser chamado em qualquer parte do codigo e faz uma tarefa e retorna um valor.

Exemplo:

<?php
$a = 120;
comp($a);
echo "$a";
function comp (&$a){
if($a <=50)$a=$a+50;
else if(($a > 50)&&($a <= 100))$a = $a +100;
else if($a > 100)$a = $a +200;
}
?>



Form ou formulario é feito em HTML e o PHP tem uma grande caracteristica que é o tratamento do mesmo.

Exemplo:

<!DOCTYPE html>
<html lang="pt-br">
<html>
<head>
<title>Exemplo</title>
</head>
<body>
<form name = "i_form" method = "POST" action="tratamento.php">
<label> Primeiro nome: </label>
<input type="texte" name="inome" size="40" maxlength="30"/>

<label> Sobrenome: </label>
<input type="texte" name="isnome" size="40" maxlength="30"/>

<label> E-mail: </label>
<input type="texte" name="iemail" size="40" maxlength="30"/>

<input type="submit" value="Enviar">
</body>
</html>



O elemento <head> é o cabeçalho do HTML.
Exemplo:

<head>
<title> Ola mundo !!</title>
</head>

O elemento <body> é o corpo do HTML, todo o conteudo vai dentro,Só pode ter um por arquivo HTML. Exemplo:

<body>
<p> Ola mundo !!</p>
</body>

Substr retorna um pedaço da string.(nome da variavel ou valor string,valor inicial,comprimento)

Exemplo:

$palavra = "teste";
$teste = substr("$palavra", 1); // retorna o valor $palavra
$teste = substr("$palavra", 2, -1); // retorna o valor "est"
$teste = substr("$palavra", -2); // retorna "te" $teste = substr('$palavra',0,-1);//retorna $palavra
